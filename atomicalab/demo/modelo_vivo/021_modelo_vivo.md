# MODELO VIVO

## Propósito

Representar el estado estimado de una organización como un organismo.

No representa departamentos.

No representa personas.

No representa procesos.

Representa el equilibrio del sistema.

---

# Componentes

El Modelo Vivo está formado por siete sistemas.

Cada sistema ocupa una posición fija.

Nunca cambia de lugar.

                PROPÓSITO

      PERSONAS          INFORMACIÓN

 FLUJO DE VALOR      GOBERNANZA

     RESILIENCIA     REGENERACIÓN

---

# Estado

Cada sistema solo puede encontrarse en uno de cuatro estados.

Estable

Evolución

Atención

Saturación

Nunca existen estados intermedios visibles.

---

# Forma

El organismo nace de la tensión entre los siete sistemas.

No existe una imagen predefinida.

Cada organización genera una geometría distinta.

La geometría se calcula.

Nunca se dibuja manualmente.

---

# Lectura

El CEO nunca interpreta números.

Observa equilibrio.

Si el organismo mantiene simetría:

equilibrio.

Si aparece una deformación localizada:

tensión.

Si aparecen varias deformaciones:

fragilidad creciente.

Si el organismo recupera simetría:

aprendizaje organizacional.

---

# Interacción

Hover

Resalta únicamente el sistema.

Click

Abre Sistema.

Nunca abre Observe directamente.

---

# Animación

El organismo permanece prácticamente inmóvil.

Solo respira.

Una respiración completa cada 10–15 segundos.

No existen efectos decorativos.

No existen partículas.

No existen pulsos continuos.

No existen animaciones gratuitas.

---

# Objetivo cognitivo

Al mirar cinco segundos el CEO debe pensar:

"La empresa está equilibrada."

o

"Algo está rompiendo el equilibrio."

Nunca necesita saber todavía qué es.
