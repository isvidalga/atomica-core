# Pregunta

¿Cuánto puedo confiar en este modelo?

Esta pantalla responde exclusivamente a esta pregunta.

No responde:

- cómo está la organización
- qué cambió
- dónde intervenir
- qué intervenciones existen

---

# Objetivo

Mostrar el grado de confianza del modelo organizacional.

No demuestra que el modelo sea verdadero.

Permite comprender qué información sostiene cada hipótesis.

---

# Usuario

CEO.

Uso previsto:

- antes de tomar decisiones importantes
- cuando desea validar una hipótesis
- cuando cuestiona una conclusión del modelo

---

# Arquitectura

Existe un único protagonista.

El nivel de confianza del modelo.

Toda la información restante explica dicho nivel.

---

# Elementos

## 1. Confianza general

Elemento principal.

Contiene:

- nivel de confianza
- tendencia
- fecha de actualización

La confianza nunca se expresa como certeza.

---

## 2. Evidencias disponibles

Cada evidencia indica:

- tipo
- origen
- fecha
- cobertura
- calidad

Las evidencias permanecen separadas de las conclusiones.

---

## 3. Cobertura

Resume qué parte del modelo dispone de evidencia suficiente.

Puede mostrar:

- Alta cobertura
- Cobertura parcial
- Cobertura insuficiente

---

## 4. Vacíos de información

Identifica únicamente aquello que el modelo desconoce.

Ejemplos:

- Falta información financiera.
- No existen entrevistas recientes.
- No hay evidencia documental.

Los vacíos nunca se ocultan.

---

## 5. Evolución de la confianza

Describe cómo cambia la confianza del modelo.

Puede aumentar.

Puede disminuir.

Puede permanecer estable.

# Principios

Toda afirmación necesita evidencia.

Toda evidencia posee calidad diferente.

Ausencia de evidencia no implica falsedad.

Mayor evidencia aumenta confianza.

Nunca elimina incertidumbre.

ATÓMICA estima.

No certifica.

¿Qué información lo sostiene?

¿Qué conclusiones son todavía débiles?
