# Pregunta

¿Qué estoy haciendo para fortalecer mi organización?

Esta pantalla responde exclusivamente a esta pregunta.

No responde:

- cómo está la organización
- qué relaciones existen
- qué cambió
- dónde intervenir
- cuánta evidencia soporta el modelo

---

# Objetivo

Registrar las intervenciones decididas por la organización y permitir observar su evolución.

No gestiona proyectos.

No sustituye herramientas como Jira, Asana o Monday.

No administra tareas.

---

# Usuario

CEO.

Uso previsto:

- revisar iniciativas activas
- comprobar evolución
- decidir continuidad, ajuste o cierre

---

# Arquitectura

Existe un único protagonista.

La intervención prioritaria.

El resto de intervenciones aparecen ordenadas por relevancia.

---

# Elementos

## 1. Intervención principal

Contiene:

- nombre
- objetivo
- estado
- fecha de inicio
- responsable
- horizonte temporal

---

## 2. Estado

Cada intervención puede encontrarse en uno de cuatro estados.

- Propuesta
- En ejecución
- En observación
- Finalizada

No existen porcentajes de progreso.

---

## 3. Observación del sistema

Cada intervención muestra:

- comportamiento esperado
- comportamiento observado
- variaciones detectadas

El objetivo es observar la respuesta del sistema.

No medir productividad.

---

## 4. Riesgo

Cada intervención indica:

- riesgo bajo
- riesgo medio
- riesgo alto

Puede incluir:

- efectos secundarios observados
- efectos no esperados

---

## 5. Resultado provisional

Resume:

- evolución favorable
- evolución neutra
- evolución desfavorable

Nunca declara éxito definitivo.

# Principios

Una intervención es un experimento organizacional.

El sistema necesita tiempo para responder.

La ausencia de cambios también es información.

Toda intervención puede producir consecuencias inesperadas.

Cerrar una intervención no implica resolver el problema.

---

# Criterios de aceptación

Después de utilizar esta pantalla un CEO debe responder:

¿Qué intervenciones siguen activas?

¿Cuál merece atención?

¿Qué cambios está produciendo?

¿Debo continuar, modificar o detener alguna?
