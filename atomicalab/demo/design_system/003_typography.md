# Filosofía

La tipografía comunica antes que el color.

Toda la interfaz debe poder funcionar únicamente en blanco y negro.

La jerarquía se consigue mediante tamaño, peso y espacio.

Nunca mediante efectos.

# Principios

Nunca usar más de cuatro tamaños distintos en una misma pantalla.

Nunca mezclar dos familias tipográficas.

Nunca usar texto justificado.

Nunca abusar de negritas.

El blanco también forma parte de la tipografía.
