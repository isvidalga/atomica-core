# Context

## Principio

Los hechos no existen aislados.

Todo conocimiento necesita contexto.

## Contexto organizacional

Sector.

Mercado.

Tamaño.

Madurez.

Ubicación.

Cadena de valor.

## Contexto temporal

Momento.

Duración.

Secuencia.

Ritmo.

## Regla

Una evidencia sin contexto pierde capacidad explicativa.
