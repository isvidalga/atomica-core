# Cognitive Objects

## Objetos fundamentales

- Observación
- Evidencia
- Afirmación
- Conocimiento
- Patrón
- Hipótesis
- Riesgo
- Decisión
- Resultado
- Aprendizaje
- Informe

## Propiedades comunes

Todos los objetos poseen:

- id
- fecha_creacion
- fecha_actualizacion
- autor
- estado
- confianza
- relaciones
- historial

## Principios

Los objetos nunca desaparecen.

Solo evolucionan.
