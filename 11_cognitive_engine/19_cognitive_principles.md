# Cognitive Principles

## Principios fundamentales

El conocimiento es acumulativo.

Toda afirmación necesita evidencia.

Toda evidencia posee contexto.

Toda decisión deja consecuencias.

Todo patrón emerge.

Toda historia permanece.

Toda inferencia puede explicarse.

Toda incertidumbre se declara.

Toda organización aprende.

Toda organización olvida.

ATÓMICA existe para reducir ese olvido.
