# Cognitive Cycle

## Principio

ATÓMICA no almacena datos.

Transforma observaciones en conocimiento accionable.

Todo elemento recorre el mismo ciclo cognitivo.

```
Observación
    ↓
Evidencia
    ↓
Afirmación
    ↓
Conocimiento
    ↓
Relaciones
    ↓
Patrones
    ↓
Hipótesis
    ↓
Decisiones
    ↓
Resultados
    ↓
Aprendizaje
    ↓
Actualización del conocimiento
```

## Reglas

- Ningún conocimiento nace sin evidencias.
- Toda evidencia puede reforzar o debilitar conocimiento existente.
- Todo patrón modifica futuras hipótesis.
- Toda decisión genera nuevos datos.
- Todo resultado alimenta el siguiente ciclo.
