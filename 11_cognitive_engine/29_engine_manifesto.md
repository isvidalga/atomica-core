# Cognitive Engine Manifesto

ATÓMICA no construye dashboards.

Construye comprensión.

No almacena documentos.

Construye memoria.

No clasifica información.

Construye conocimiento.

No calcula únicamente indicadores.

Descubre estructuras.

No automatiza decisiones.

Hace visibles sus consecuencias.

No pretende sustituir el criterio humano.

Pretende aumentarlo.

Toda organización posee datos.

Pocas poseen inteligencia.

ATÓMICA existe para cerrar esa distancia.
