# Cognitive Memory

La memoria organizacional es acumulativa.

## Conserva

Eventos.

Evidencias.

Hipótesis.

Errores.

Decisiones.

Resultados.

## Nunca conserva

Opiniones sin evidencia.

Suposiciones olvidadas.

Información sin contexto.

## Principios

La memoria nunca resume.

La memoria conserva.

La inteligencia interpreta.
