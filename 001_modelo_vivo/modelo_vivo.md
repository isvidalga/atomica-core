# Modelo Vivo

## 1. Propósito

El Modelo Vivo es la representación sistémica de una organización.

No representa departamentos.

No representa personas.

No representa procesos.

Representa el estado de un sistema vivo.

Su función es permitir que un CEO comprenda dónde debe intervenir sin reducir la organización a indicadores aislados.

El Modelo Vivo es la superficie principal de ATÓMICA.

Todo el resto del instrumento existe para explicar aquello que el Modelo Vivo revela.

---

## 2. Qué representa

El Modelo Vivo representa únicamente aquello que puede inferirse de forma razonable a partir del conocimiento disponible.

No muestra hechos.

No muestra opiniones.

No muestra predicciones.

Representa hipótesis estructurales sobre el funcionamiento de la organización.

Cada elemento visible corresponde a una estimación cuya confianza depende de la calidad y cantidad de evidencia disponible.

El Modelo Vivo nunca representa certeza.

Representa estados de conocimiento.

---

## 3. Qué no representa

El Modelo Vivo no es:

- un dashboard
- un ERP
- un organigrama
- un cuadro de mando
- un sistema ESG
- un gemelo digital físico
- una simulación

Tampoco pretende explicar la organización.

Su única misión es orientar la atención.

Toda explicación pertenece a superficies posteriores.

---

## Principio Fundamental

Una organización no puede comprenderse observando elementos aislados.

Solo puede comprenderse observando las relaciones entre ellos.

Por ello el Modelo Vivo representa relaciones antes que objetos.
